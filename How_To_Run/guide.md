# How to Run Liquid Glass 18x18 Tic-Tac-Toe

The game has been fully integrated into a modern React Vite environment with a stunning "Liquid Glass" UI. You don't need to scaffold a new project anymore.

## Prerequisites
You must have **Node.js** installed on your system. 
- You can download it at https://nodejs.org 

---

## Option 1: Automated Script (Windows)

Included in this folder is a file called `setup.bat`. 
1. Double-click the `setup.bat` file to run it.
2. The script will automatically start the development server for the existing project.
3. Wait a moment, and your browser should automatically open to `http://localhost:5173`. 

*(Note: Stop the server whenever you are done by closing the Command Prompt or pressing `Ctrl + C`).*

---

## Option 2: Step-by-Step Manual Setup

If you prefer to run it manually via the terminal:

### 1. Open your Terminal
Navigate to the frontend directory:
```bash
cd ../antigravity-project/frontend
```
*(Or navigate to `antigravity-project/frontend` from the root of the project.)*

### 2. Install Dependencies (First time only)
```bash
npm install
```

### 3. Start the Game
Run the development server using:
```bash
npm run dev
```

Click the link provided in the terminal (usually `http://localhost:5173`) to open the game in your web browser. Enjoy!
